                        Commons Lang Package
                            Version 2.7
                           Release Notes


INTRODUCTION:

This document contains the release notes for the 2.7 version of Apache Commons Lang.
Commons Lang is a set of utility functions and reusable components that should be of use in any Java environment.


COMPATIBILITY WITH 2.6
======================
Lang 2.7 is binary compatible release with Lang 2.6, containing bug fixes and small enhancements.

Lang 2.7 requires a minimum of JDK 1.3.


IMPROVEMENTS IN 2.7
===================

    * The Automatic-Module-Name MANIFEST entry has been added for Java 9 compatibility.
      The module name is "org.apache.commons.lang".

BUG FIXES IN 2.6
================

    * [LANG-807] - RandomStringUtils throws confusing IAE when end <= start
    * [LANG-805] - RandomStringUtils.random(count, 0, 0, false, false, universe, random) always throws java.lang.ArrayIndexOutOfBoundsException
    * [LANG-803] - LocaleUtils - DCL idiom is not thread-safe
    * [LANG-677] - DateUtils isSameLocalTime() compares the hour using 12hour Calendar.HOUR instead of 24hour Calendar.HOUR_OF_DAY
